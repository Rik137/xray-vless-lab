## xray-vless-lab  
---
### Описание  
Этот проект предназначен для пользователей, которым требуется надёжный и воспроизводимый способ обхода интернет-цензуры, DPI и сетевых ограничений.  
Репозиторий содержит практическую инструкцию по развёртыванию и базовой настройке Xray Core с использованием протокола VLESS. Основной фокус сделан на   стабильную работу в условиях активных блокировок, минимализм конфигурации и понимание происходящего, а не слепое копирование конфигов.  

Проект носит учебно-практический характер и предназначен для самостоятельного развёртывания на собственном сервере.  

### Требования  
Перед началом убедитесь, что выполнены следующие условия:  
Арендован VPS в юрисдикции без агрессивной интернет-цензуры  
(рекомендуется Европа, Азия, Латинская Америка)  
Операционная система: Ubuntu 20.04+  
Доступ к серверу по SSH с правами root или sudo 
Базовое понимание работы Linux и сетевых сервисов  

### Установка Xray  
Установка производится из официального репозитория.  
```bash
sudo apt update
sudo apt install xray
```
После установки бинарный файл будет доступен по пути:  
```bash
/usr/bin/xray
```

### Запуск и автозагрузка  
Запустите сервис Xray и добавьте его в автозагрузку:  
```bash
sudo systemctl start xray
sudo systemctl enable xray
```
Проверьте статус сервиса:  
```bash
sudo systemctl status xray
```
Сервис должен находиться в состоянии active (running).  

### Конфигурация  
Расположение конфигурационного файла  
Основной конфигурационный файл Xray должен находиться по пути:  
```bash
/usr/local/etc/xray/config.json
```
В рамках этого проекта предполагается использование уже подготовленного конфигурационного файла server-config.json.  
Скопируйте его под именем config.json в указанный каталог.  
Пример:
```bash
vim config_client.json /usr/local/etc/xray/config.json
```
всатвьте скопируемый файл  
После изменения конфигурации обязательно перезапускайте сервис:  
```bash
sudo systemctl restart xray
```
### Генерация ключей и UUID  
Для работы протокола VLESS требуется уникальный UUID клиента.  
Сгенерировать UUID можно с помощью встроенной команды Xray:
```bash
xray uuid
```
Пример вывода:
```bash
e7b0c2a4-9f3d-4b6a-8d91-xxxxxxxxxxxx
```

Проверка работы
После запуска и настройки:  
Убедитесь, что Xray работает без ошибок:  
```bash
journalctl -u xray -e
```
Проверьте, что порт прослушивается:  
```bash
ss -tulnp | grep xray
```
Подключите клиент и проверьте доступ к заблокированным ресурсам  
Если соединение не устанавливается — смотрите логи, а не гадайте.  
Безопасность и замечания  
Не используйте стандартные порты без необходимости  
Ограничивайте доступ к серверу с помощью ufw или iptables  
Не храните приватные конфиги в публичных репозиториях  
Регулярно обновляйте систему и Xray  
Проект не предоставляет готовый VPN-сервис, а лишь инструменты и понимание для самостоятельного развёртывания.
