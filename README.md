# xray-vless-lab  
---
Этот проект предназначен для людей, которым требуется надёжный способ обхода интернет-цензуры и сетевых ограничений.
В репозитории представлена практическая инструкция по развёртыванию и использованию экосистемы Xray (VLESS), ориентированная на стабильную работу в условиях блокировок и DPI.

---
## Требования
1. Арендовать VPS в юрисдикции без агрессивных интернет-блокировок
2. поднчть на нем движок xray

```bash
sudo apt update
sudo apt install xray
```
3. запустить xray и добавть его в автозагрузку
   
 ```bash
sudo systemctl start xray
sudo systemctl enable xray
```
4. проверить запуск xray
```bash
sudo systemctl status xray
```
5. положить уже готовай config file c именем config_client.json по пути

```bash
/usr/local/etc/xray/config.json
```
6. сгенрировать ключи доступа при помощи команды

```bash
xray
